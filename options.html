⚙️ Settings

Configure your GitHub organizations and preferences

GitHub Organizations

Enter the organization names you want to search (one per line)

Organization names should match the GitHub URL: github.com/org-name/repo-name

GitHub Personal Access Token (Optional)

Add a token for higher API rate limits (5000 requests/hour vs 60 without)

Create a token at github.com/settings/tokens.
For public repos only: Select public_repo scope.
For private repos: Select full repo scope.

Cache Duration

How long to cache repository data before refreshing

Longer cache times reduce API calls but may show outdated data

Keyboard Shortcut

The default shortcut is Ctrl+Shift+G (or Cmd+Shift+G on Mac).

To customize it, visit chrome://extensions/shortcuts
